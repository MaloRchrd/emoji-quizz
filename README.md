![emoji-quizz](/public/img/emoji-quizz.png)

## stack technic
  - Node
  - Express
  - Socket.io
  - MongoDB

## front-end
  - jQuery
  - Bootstrap
  - List.js
  - pug.js

## Instalation

- Clone repository on your desktop
- if not installed,[add MongoDB](https://docs.mongodb.com/manual/installation/)
- Start mongo
- open terminal :

```
cd /pathToFolder/emoji-quizz
npm install
node index.js
```
